Process(프로세스)

프로그램이 실행될 때 운영체제에서 메모리 공간을 할당받아 실행
각 프로세스는 독립된 메모리 공간을 가지고 있음
다른 프로세스와 메모리 영역을 공유하지 않음

프로세스와 스레드의 관계
프로세스는 스레드를 포함한다 : 하나의 프로세스는 적어도 하나의 스레드(작업)을 포함
스레드(작업)는 그 프로세스(작업실행하는 공간)에서 실행되는 프로그램의 실행 흐름을 담당

하나의 프로세스의 여러 스레드 : 여러 스레드 (여러 작업)가 하나의 프로세스(공간)내에서 동시에 각자의 작업을 실행하고, 이 스레드들(작업들)은 서로 맡은 작업을 공유하고 동시에 작업을 수행할 수 있음

프로세스 : 독립적으로 실행되는 프로그램의 인스턴스, 각기 다른 메모리 공간을 가짐
스레드   : 프로세스 네에서 실행되는 작은 단위로, 프로세스 자원을 공유하며 동시에 여러 작업을 처리 할 수 있음

프로세스 : 하나의 작업 단위
            컴퓨터로 예를 들면
            1번 컴퓨터 = 멀티프로세스 , 단일스레드(메모장, 크롬, 그림판이 하나의 작업만 진행하고 있기 때문)
                메모장     크롬  그림판
            2번 컴퓨터 = 단일프로세스 , 단일스레드
                메모장
            3번 컴퓨터 = 멀티프로세스, 단일스레드 , 멀티스레드(크롬은 10가지, 그림판은 2가지 작업을 동시에 멀티로 진행하고 있기때문)
                메모장     크롬(10)  그림판(2)
